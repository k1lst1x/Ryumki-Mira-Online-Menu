# Ryumki Mira Online Menu
Онлайн-меню для бара «Рюмки на Мира» (Алматы), разработанное на Django с поддержкой многоязычности и адаптивным UI.

## Локальный запуск (Windows)

```powershell
py -m venv .venv
. .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
